# Belgravia Ace website

A standalone, static property website prepared for GitHub Pages. It has no build step and no dependency on Weebly.

## Preview locally

From this folder, run:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Publish with GitHub Pages

1. Create a GitHub repository and add the contents of this folder to its root.
2. In the repository, open **Settings → Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select the `main` branch and `/ (root)`, then save.

The enquiry form opens WhatsApp with the visitor's details pre-filled. Update the phone number `6597995300` in `index.html` and `assets/script.js` if the sales contact changes.

Before launch, confirm current pricing, availability, expected TOP, developer details and all regulatory copy with the appointed sales team.
